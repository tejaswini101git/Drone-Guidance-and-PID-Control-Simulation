# Drone Guidance and PID Control Simulation

## Overview

This project demonstrates PID control and guidance algorithms for autonomous systems using Python simulations.

The project includes:

- Drone altitude control using a PID controller
- Wind disturbance handling
- Autonomous boat path tracking
- Boat guidance under environmental current disturbances

## Technologies

- Python
- Google Colab
- NumPy
- Matplotlib
- ipywidgets

## Results

### Drone PID Control
- Tuned P, I and D gains
- Wind disturbance analysis
- Stable altitude tracking

### Boat Guidance
- Path tracking without disturbance
- Path tracking with current disturbance
- Animated boat guidance

## Skills Demonstrated

- PID Controller Tuning
- Feedback Control
- Autonomous Navigation
- Guidance Systems
- Python Simulation

## Note

This project was completed as part of an internship assignment using the provided simulation notebook. The work includes controller tuning, simulation analysis, and interpretation of the results.
